# Бабусині іграшки — дизайн-хендофф (для збірки на Astro)

Документ описує дизайнерські рішення сайту. Джерело істини щодо верстки — `Бабусині іграшки.dc.html` (десктоп + мобільні макети).

---

## 1. Бренд і тон
- **Назва:** Бабусині іграшки. **Тег-лайн:** «М'які іграшки ручної роботи».
- **Продукт:** авторські іграшки, **в'язані спицями** з натуральної пряжі (не шиті).
- **Майстриня:** Світлана.
- **Настрій:** тепла ностальгія, рукотворність, затишок.
- **Уникати:** емодзі, AI-градієнтів, стокової «водички», зайвих цифр/іконок. Мінімалізм із теплом.

---

## 2. Кольори (oklch)
| Роль | Значення |
|---|---|
| Фон сторінки / секцій | `oklch(96–97% 0.012 80)` — тепла кремова біла |
| Текст основний | `oklch(30–33% 0.045 50)` |
| Текст приглушений | `oklch(45–50% 0.03 55)` |
| Акцент (теракота) | `oklch(58% 0.14 40)` |
| Акцент hover / тінь | `oklch(46% 0.13 40)` |
| Підкладка секції «Як замовити» | `oklch(93% 0.022 72)` |
| Картка (біла) | `oklch(99% 0.006 85)` |

**Бейджі статусу** (білий текст, pill):
- «В наявності» — `oklch(45% 0.09 145)` (зелений)
- «На замовлення» — `oklch(55% 0.12 80)` (бурштин)
- «Продано» — `oklch(55% 0.015 60)` (сірий); картка притлумлена `opacity:0.55`, без hover

Акцент теракота використовується для кнопок, посилань, цифр кроків, eyebrow-підписів.

---

## 3. Типографіка
- **Заголовки:** Comfortaa (700).
- **Текст / кнопки:** PT Sans (400 / 700).
- **Масштаб (десктоп):** H1 ~46px · H2 секцій ~30px · назва картки 16px · ціна 17px bold · eyebrow 13px.
- **Eyebrow:** дрібний uppercase теракотою, `letter-spacing:0.08em`, відкриває кожну секцію.

---

## 4. Компоненти й патерни
### Кнопки
- Форма pill: `border-radius:100px`.
- Первинна — заливка теракотою, білий текст.
- Вторинна — контур теракотою (прозорий фон).
- Мікро-анімація: `hover → translateY(-2px)`, `active → translateY(1px)`.

### Картки каталогу
- Біла, `border-radius:16px`, м'яка тінь.
- Фото зверху **3:4**, `object-fit:cover`.
- Бейдж статусу — абсолютно, лівий-верхній кут фото.
- Нижче: назва, розмір (см), ціна (грн).
- **Hover:** `translateY(-6px) rotate(-1deg)` + глибша тінь. Проданим — без hover.

### Сітки
- `display:grid` + `gap` (не inline-flow). Каталог — 4 колонки; кроки — 3 колонки.

---

## 5. Hero
- Повнокадрове фото пряжі на весь блок, **майже без вуалі** (фото соковите, як оригінал).
- Поверх — радіальна кремова вуаль лише для згасання країв:
  `radial-gradient(ellipse 90% 130% at 50% 45%, cream/0.14 0%, cream/0.5 60%, cream/0.95 100%)`
  (де `cream = oklch(97% 0.012 80)`) — кути плавно зливаються з кремовим фоном.
- Текст **темний, по центру**, з м'яким **світлим text-shadow** для читабельності над строкатими нитками:
  - H1: `text-shadow:0 2px 12px oklch(98% 0.008 85 / 0.85)`
  - eyebrow / параграф: `text-shadow:0 1px 8px oklch(98% 0.008 85 / 0.92)`
- Дві CTA: «Переглянути каталог» (первинна), «Написати в Telegram» (вторинна).

---

## 6. Фон сторінки
- Те саме фото пряжі як тло всієї сторінки: `background: url(...) center/cover`, `filter: blur(~4–8px)`, легкий `transform: scale(1.05–1.12)`.
- Під контентом — радіальна кремова вуаль (центр ~0.5 → кути ~0.97), щоб текст і білі картки завжди лишались читабельними.

---

## 7. Структура сторінок
### Головна
Header → Hero → Каталог → «Як замовити» (3 кроки) → Footer.

### Про майстриню (окрема сторінка)
Header → фото Світлани (портрет 4:5) + біо (2–3 абзаци) + перелік переваг (15+ років · натуральні матеріали · ручна робота) + CTA «Переглянути каталог» → Footer.

### Сторінка іграшки
Галерея (360° / мініатюри) → назва → ціна + «в обране» (сердечко з анімацією heart-beat) → характеристики (розмір, матеріали, статус) → кнопки замовлення (Telegram / Viber).

### Header
Лого + тег-лайн зліва · nav (Каталог, Про майстриню, Як замовити, Контакти) · кнопка «Замовити» справа. Активний пункт nav — теракотою.

### Footer
Контакти (Telegram / Viber / Instagram) + умови доставки Новою поштою (2–3 дні після підтвердження).

---

## 8. Нотатки для Astro
- Реальні фото товарів — вертикальні **3:4**, тримати `object-fit:cover`.
- Nav-пункти — справжні лінки між сторінками; активний підсвічувати теракотою.
- Природний розклад на компоненти: `Header`, `Hero`, `CatalogGrid` + `ToyCard`, `Steps`, `MasterAbout`, `Footer`.
- Статуси товару винести в дані (enum: `in_stock` / `made_to_order` / `sold`) → мапити на колір бейджа + поведінку картки.
- Товари можна тримати в `content collections` (назва, розмір, ціна, статус, фото).
- Кремова вуаль над фоном/hero — це overlay-шар (`position:absolute; inset:0`), не `background-blend`; зручно винести в утиліту/клас.
